BI e um conjunto de tecnicas e processos para coletar, tratar, armazenar, analisar, compartilhar e monitorar os dados

estudos adicionais: principio de pareto, KPI, balanced score card

passo 1 - etapa de coleta de dados nas fontes de dados(banco de dados relacionais,planilhas,emails,etc)

passo 2 - organizar esses dados obtidos na etapa 1 em um unico lugar(data warehouse)

passo 3 - etapa de analise e compartilhamento desses dados obtidos na etapa 1 e analisados na etapa 2

passo 4 - se obtem uma decisao com base nos dados que foram coletados, analisados e armazenados nas etapas anteriores

passo 5 - etapa de monitoramento dos dados e das decisoes tomadas em decorrencia de todo o processo anterior, para se verificar se as decisoes tomadas trouxeram 
	  bons resultados para o negocio.

perguntas que o BI responde: o que aconteceu? / quantas vezes isso aconteceu?  / com que frequencia isso aconteceu ou acontece ? 


****** ETAPAS NO PROCESSO DE CONSTRUCAO DE UMA SOLUCAO DE BI *******

PASSO 1 - mapeamento das origens dos dados(data sources), que sao os locais onde os dados brutos estao armazenados, tambem nessa etapa fazemos o mapeamento do formato
	  em que esses dados estao armazenados nas fontes

PASSO 2 - apos mapear todos os dados e suas origem comeca o processo de ETL ou ELT, nesse processo conectamos nas fontes dos dados com alguma ferramenta de ETL
	  extraimos os dados das fontes e colocamos na staging area que é onde faremos as tranformacoes nesses dados, ou seja, trabalhamos nos dados para que eles
	  fiquem no formato adequado para serem carregados na DW, apos a etapa de tranformacao e com os dados ja no formato correto e unificados eles sao carregados 
	  no DW e disponibilizados para o consumo dos usuarios.


PASSO 3 - Nessa etapa é construindo o DW, sao definidas as tecnologias de armazenamento, se sera on primese ou se sera na nuvem, onde o DW sera construindo, depois 
	  é feita a modelagem dimensional do DW e trabalhado na sua construcao e conexao com a area de staging.


PASSO  4 - Essa e a etapa de analise, onde sao usadas tecnicas de analise de dados e estatistica para realizar a analise na massa de dados que esta armazenada no DW
	   afim de gerar conhecimento para os tomadores de decisao. EX:. Metricas,KPI,analisys OLAP


PASSO 5 - Nessa etapa sao definidas as formas de visualizacao dos dados que seram entregues para os usuarios de negocio do BI, REPORTING,DASHBOARD,
