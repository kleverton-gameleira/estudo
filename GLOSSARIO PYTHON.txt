python --version  = este comando retorna a versao do python que esta instalada no sistema operacional

pip --version = retorna a versão do pip(gerenciador de pacotes) que esta presente na instalacao atual do python

python -m pip install --upgrade pip  = este comando faz a atualizacao da versao do pip instalado no sistema operacional pela versao mais recente lancada

****** TRABALHANDO COM AMBIENTES VIRTUAIS *********
 
1 passo = acessar a pastas onde estao os arquivos de pip e lib e copiar o caminho EX: c:/user/nome_user/AppData/Local/programs/Python/python-32/Scripts

2 passo = adicionar um path com o caminho copiado

3 passo criar uma variavel de ambiente e como o nome WORKON_NOME e o valor da variavel %USERPROFILE%\Envs

virtualenv: ambiente virtual na linguagem python que abstrai a instalação padrao do python

pip install virtualenv  = este comando instala o ambiete virtual no python que abstrai a instalacao padrao do python

pip install virtualenvwrapper-win  = instala ambiente virtual no python

mkvirtualenv nome_ambiente  = este comando cria um ambiente virtual para se trabalha com o python de maneira isolada

deactivate = este comando sair do ambiente virtual e volta para o ambiente de padrao do python no sistema operacional

workon nome_ambiente  = este comando entra no ambiente virtual criado.

rmvirtualenv nome_ambiente  = este comando exclui um ambiente ja existente
